about this website

technologies

this is a mostly-static site built using Eleventy and serverless functions. it is currently hosted on Netlify.

javascript is only used for non-essential enhancements like the panel. the menubar and pop-up windows utilize the Popover API.

design

this website’s visual language is based on the user interfaces of Macintosh System 7 and OS 9 and is specifically inspired by the work of Susan Kare and Bill Atkinson. most of the and interface elements are pulled from classic Mac system software, apps, and fonts.

fonts

Archivo is used for headings, Spline Sans for body text, and Spline Sans Mono for code and some UI elements, all hosted locally.
